click_extra.config.load_conf(ctx, param, conf_path, strict=False)[source]#

Fetch parameters values from configuration file and merge them to the defaults.

User configuration is merged to the context default_map, as in: https://click.palletsprojects.com/en/8.0.x/commands/#context-defaults

This allow user’s config to only overrides defaults. Values sets from direct command line parameters, environment variables or interactive prompts takes precedence over any values from the config file.
